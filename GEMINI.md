# Smart Kitchen Furniture Project

This is a Next.js application for a custom kitchen furniture and modern railing fabrication business.

## Project Overview
- **Core Business:** Custom on-site fabrication of high-end Kitchen Trolleys and Modern Railings.
- **Technology Stack:**
  - Next.js (App Router)
  - React
  - TypeScript
  - Tailwind CSS

## Conventions
- **Styling:** Use Tailwind CSS for styling.
- **Components:** Place shared components in `components/`.
- **Pages:** Define routes within the `app/` directory using the App Router.
- **Assets:** Place static assets (images, icons) in the `public/` directory.

## Development Workflows
- **Running locally:** `npm run dev`
- **Building:** `npm run build`
- **Linting:** `npm run lint`
